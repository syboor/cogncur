The glyphs cgi and cgt1 are interchangeble. If one of these glyphs 'fits' as entry strokes towards a letter, the other glyph will also fit. Stylistic set ss09 changes (among other things) cgt1 to cgi to realize this versions of the letter t: t. Stylestic set ss12 changes cgi to cgt1 to achieve 'undercurve' connections: towards i j p w p.

Note that cgi and cgn are not interchangeable. To use a different entry stroke, a different variant of the letter glyph will be needed. This is why the letters r, w, y and z all have multiple variants.

The glyphs cgz and cgn are interchangeble, but cgz has a slope that better matches the internal diagonal stroke of the letter z. The same is true for cgz1 and cgi

Short and invisible entry strokes

With stylistic set ss01, entry strokes from the baseline will be replaced by the set of entry strokes below: ...

Some of these glyphs do not contain visible strokes, but are used for their effect on spacing, either through whitespace contained in the glyph, or through entries in the kerning tables for the glyph. Lowercase letters that are not proceeded by another lowercase letter must be proceeded by an entry stroke glyph. That is the case even for letters like a that do not have an actual entry stroke. They still need an entry stroke glyph, because these glyphs are used in the kerning tables of the font to achieve correct distance between interpunction (such as: opening quotes, the Spanish upside down question mark) and the letter. Lowercase letter glyphs are never used in our kerning tables.
